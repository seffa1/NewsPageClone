## To start the react app

In the project directory, you can run:
### `npm start`
This fictional news site was cloned from a design on Dribble.com. I used the News API to feed live news articles, sorted by topic. The live site is currently using historical data as the live API cost ~$600 a month. See config.js about how to switch between local API usage, or loading of old data.
